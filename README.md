# MR Foods Website

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm/yarn

### Installation

```bash
pnpm install
# or
npm install
```

### Development

```bash
pnpm dev
# or
npm run dev
```

### Production Build

```bash
pnpm build && pnpm start
# or
npm run build && npm start
```
